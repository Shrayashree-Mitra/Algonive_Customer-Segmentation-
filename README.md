# Algonive_Customer-Segmentation
Customer Segmentation System using Python — Clustering algorithms, EDA, and data visualization to categorize customers based on purchasing behavior
Customer Segmentation Using Python -
 Overview:
A Customer Segmentation System that categorizes customers based on their purchasing behavior, demographics, and preferences using Python and Machine Learning techniques.
Objective:
To help businesses target marketing strategies, personalize recommendations, and optimize customer engagement by grouping similar customers together.
Technologies Used:
Python (Pandas , NumPy , Scikit-learn)
Matplotlib / Seaborn
Power Bi
Key Features:
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Customer Segmentation using Clustering (K-Means)
Data Visualization of segments
Feature Engineering
The final features selected for clustering are Age, Annual Income (k$), Spending Score (1–100), and Gender_Encoded. These features capture both demographic and behavioral aspects of customers
Feature Ranges:
Age: 18 – 70
Annual Income: 15k – 137k
Spending Score: 1 – 99
Clustering (K-Means) -
Optimal Clusters:
Best K = 10
Silhouette Score = 0.4208
Key Insights :
Identified high-value customers (VIPs) with high income & spending
Found low-income high spenders (Impulse Buyers) → potential risk segment
Detected high-income low spenders (Savers) → opportunity for targeted marketing
Majority customers fall under moderate spending behavior
Conclusion :
This project demonstrates how customer data can be effectively segmented using K-Means clustering. By applying data preprocessing, feature engineering, and analysis, distinct customer groups were identified based on income and spending behavior. The results provide actionable insights, such as identifying high-value customers, potential growth segments, and risk-prone buyers. Overall, the project shows how machine learning can support data-driven marketing and better customer targeting.
